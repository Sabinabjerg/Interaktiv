Phishing

Falske mails eller beskeder, som oftest lokker modtageren til at klikke på et link eller dele følsomme oplysninger. Dette er ofte første skridt i et cyberangreb

Malware

En ondsindet software, som kan slette data, stjæle oplysninger eller spionere brugen. Malware kan gemme sig i links, filer eller programmer.

Ransomware

Er en type malware, som krypterer data og kræver løspenge for at genskabe adgangen. De holder altså data som gidsel.
